# Интернет-магазин косметики на Django

Этот проект представляет собой интернет-магазин косметики, разработанный на Django. Основная идея проекта - создать удобную систему для работы с каталогом товаров, оформления заказов и управления ассортиментом.

Особенностью проекта является его интеграция с Telegram-ботом. Сайт и бот используют общие данные о товарах и остатках, благодаря чему изменения, внесённые через одну систему, доступны и в другой.

## Возможности проекта

- Регистрация и авторизация пользователей.
- Каталог товаров с категориями и подкатегориями.
- Страница товара с подробным описанием и изображениями.
- Корзина и оформление заказа.
- Личный кабинет пользователя.
- Административная панель для управления товарами.
- Загрузка и хранение изображений товаров.
- Интеграция с Telegram-ботом для работы с ассортиментом и заказами.
- Общая база данных между сайтом и Telegram-ботом.

## Используемые технологии

- Python
- Django
- HTML5
- CSS3
- JavaScript
- SQLite
- Git
